# AI Chat Assistant - Frontend Engineer Assessment

A modern, production-ready AI chat application built with React and Express.js. This application demonstrates a complete frontend-backend integration with OpenAI's GPT model, featuring conversation memory, real-time interactions, and a clean user interface.

## Features

### Core Requirements
- **Prompt input** with validation and character counting
- **AI API integration** with OpenAI GPT-3.5 Turbo
- **Dynamic response display** with message bubbles
- **Comprehensive error handling** with user-friendly messages
- **Loading states** with visual indicators
- **Responsive design** that works on mobile and desktop

### Bonus Features (+10 points)
- **Conversation history** with full context memory
- **Clear button** to reset conversation
- **Auto-scroll** to latest message
- **Enter-to-submit** functionality (Shift+Enter for new line)
- **Message differentiation** (user vs AI with distinct styling)
